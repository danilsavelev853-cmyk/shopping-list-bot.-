# Shopping List + Notes + Reminders Bot

## Что это
Telegram-бот на aiogram 3: список покупок, заметки, напоминания на конкретную дату и время. Всё работает локально, без обращений к внешним ИИ-сервисам — единственный нужный ключ это токен самого бота.

Понимает команды в свободной форме через обращение «Марк»:
- «Марк добавь молоко в список покупок»
- «Марк удали молоко»
- «Марк напомни мне в 18:00 дата 09.07.2026 позвонить клиенту»
- «Марк запиши купить подарок»
- «Марк покажи список / заметки / напоминания»
- «Марк очисти список»

Плюс обычное меню с кнопками для тех же действий.

## Переменные окружения (Render → Environment)
- `BOT_TOKEN` — токен от @BotFather (единственная обязательная)
- `TZ_OFFSET_HOURS` — опционально, сдвиг часового пояса от UTC для напоминаний. По умолчанию `5` (Уфа/Башкортостан, круглый год без перехода на летнее время)

## Деплой на Render (Web Service, бесплатный тариф)
Background Worker на Render всегда платный. Используем Web Service — бот работает через polling, рядом крутится фиктивный HTTP-сервер, отвечающий "ok" на пинг.

1. render.com → New → **Web Service**
2. Connect a repository → выбери репозиторий с этим кодом
3. Настройки:
   - Runtime: Python 3
   - Branch: `main`
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python bot.py`
   - Instance Type: **Free**
4. Environment Variables: добавь `BOT_TOKEN` (и `TZ_OFFSET_HOURS`, если нужен другой часовой пояс)
5. Deploy Web Service

## UptimeRobot — чтобы сервис не засыпал
Free Web Service на Render засыпает после 15 минут без запросов.
1. uptimerobot.com → New Monitor → HTTP(s)
2. URL: адрес сервиса на Render (вида `https://имя-сервиса.onrender.com`)
3. Interval: 5-14 минут

## Хранилище
SQLite-файл `assistant.db` рядом с bot.py. На Render диск эфемерный — при передеплое всё обнулится. Для постоянности между деплоями — Render Disk или Postgres, скажи, докручу.

## Частые проблемы
- **`TelegramConflictError` в логах на пару секунд** — нормально, это момент передеплоя (старый инстанс глушится, новый поднимается). Если продолжается дольше минуты — проблема.
- **Бот не отвечает** — проверь Logs на Render, скорее всего неверный `BOT_TOKEN` или воркер не поднялся.
- **Напоминание пришло не в то время** — проверь `TZ_OFFSET_HOURS`, по умолчанию UTC+5.
